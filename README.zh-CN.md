# $ISNAD

**AI 智能体的信任层**

一种面向智能体互联网的权益证明（Proof-of-Stake）审计协议。审计者通过质押代币为代码安全性背书。恶意代码将导致质押被销毁，而安全的代码则赚取收益。

## 问题

AI 智能体常从不可信的来源安装技能（Skills）。一个恶意技能就能窃取凭据、外泄数据或破坏系统。目前缺乏标准化的信任评估方法。

## 解决方案

**权益证明审计：**
- 审计者质押 $ISNAD 为技能背书
- 若发现恶意软件，质押将被销毁
- 安全技能将为审计者赚取收益
- 用户在安装前检查信任评分

## 词源

*Isnad* (إسن端) — 阿拉伯语，意为“支持链”。这是伊斯兰学术界通过追踪传承链来验证圣训（hadith）真实性的传统。一段言论的可靠性取决于其传承者的可靠性。

$ISNAD 将这一古老智慧应用于代码溯源。
